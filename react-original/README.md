# React Original Implementation

This folder contains the original React/TypeScript implementation of Circuit Forge Play.

## What's Here:
- **React Components**: TypeScript components for the web interface
- **Vite Build System**: Modern frontend build tooling
- **Tailwind CSS**: Utility-first CSS framework
- **Component Library**: shadcn/ui components

## Current Status:
This React implementation has been **replaced** by a Streamlit version for easier deployment and maintenance.

## Files:
- `src/` - React TypeScript source code
- `public/` - Static assets
- `package.json` - Node.js dependencies
- `vite.config.ts` - Vite configuration
- `tailwind.config.ts` - Tailwind CSS configuration
- `README_REACT.md` - Original React README

## To Run React Version:
```bash
cd react-original
npm install
npm run dev
```

**Note**: The active implementation is now the Streamlit app in the parent directory (`app.py`).
